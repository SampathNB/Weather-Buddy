## Run Locally

Clone the project

```bash
  git clone https://github.com/SampathNB/Weather-Buddy.git
```

Go to the project directory

```bash
  cd Weather-Buddy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
